## 💾 Installation
To run the scripts locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/MakeMePBaJ/Python.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Python
   ```
3. Run any Python script:
   ```bash
   python script_name.py
   ```
